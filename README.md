# landing-page-grafik-20260210

Responsywna strona landing dla grafika komputerowego — gotowy, prosty szablon w HTML/CSS/JS.

Co zawiera:
- index.html — strona główna z sekcjami: hero, usługi, portfolio, o mnie, kontakt
- styles.css — nowoczesne, ciemne style z akcentem
- script.js — proste zachowanie formularza (symulowane wysłanie) i aktualizacja roku

Szybkie uruchomienie:
1. Sklonuj repo: git clone https://github.com/arturwyroslak/landing-page-grafik-20260210.git
2. Otwórz index.html w przeglądarce.

Dalsze kroki / Customizacja:
- Podmień zdjęcia i teksty w index.html
- Podłącz formularz do backendu (Netlify Forms, Formspree, lub własny endpoint)
- Dodaj optymalizację obrazów i CI (np. GitHub Actions) przed wdrożeniem
